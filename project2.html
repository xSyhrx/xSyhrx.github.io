C# Custom Built Math Libary

Vectors

In my library, I've created a fundamental tool called Vec2 for working with 2D vectors. It's a simple and powerful structure that can be used to do things like normalize vectors, find their lengths, and easily perform operations like addition and subtraction. It's a versatile and user-friendly tool for anyone dealing with 2D vectors in programming and graphics.

Matrices

In this math library, I've introduced the Mat3 struct for working with 3x3 matrices. It simplifies the creation of transformation matrices for tasks like translation, rotation, and scaling. You can effortlessly construct transformation matrices with the provided static methods, such as CreateTransform, CreateTranslation, CreateScale, and CreateRotation, making it easy to manipulate 2D graphics and geometric transformations. Additionally, the struct offers accessor properties for extracting and setting translation, scale, and rotation components.
